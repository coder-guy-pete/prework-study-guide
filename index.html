Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: ensures that my local main branch is up to date with the main branch in Github
  • git checkout main: Switches me to the main branch on my local machine
  • git add -A: adds all of the changes I have made to my next commit
  • git commit -m "inset commit comment": Commits the changes I have made as a new commit snapshot
  • git push origin branch-name: Transmits local branch commits to the remote repository
